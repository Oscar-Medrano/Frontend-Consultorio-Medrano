# Frontend - Consultorio Medrano

Esta carpeta contiene el cliente web desarrollado con **Vue.js** y **Tailwind CSS**.

## Scripts Disponibles

En este directorio, puedes ejecutar:
- `npm run dev`: Inicia el servidor de desarrollo.
- `npm run build`: Genera el bundle para producción.
- `npm run preview`: Previsualiza la compilación localmente.
