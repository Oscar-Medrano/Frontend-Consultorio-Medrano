# 🏥 Sistema de Gestión - Consultorio Medrano

**Fullstack Developer Solution** | Next-gen Medical Management

Este repositorio contiene el **Frontend** de una plataforma profesional diseñada para optimizar la gestión de consultas médicas. Construido con un enfoque en alto rendimiento y escalabilidad.

## 🛠️ Tech Stack

- **Frontend:** Vue.js (SFC), JavaScript (ES6+), Tailwind CSS.
- **Backend (Private):** Robust API built with **Golang**.
- **Database:** Advanced Database Design (PostgreSQL).
- **Architecture:** Restful API, JWT Auth, Concurrent Data Processing.

## 🚀 Características Principales

- **Dashboard de Gestión:** Visualización en tiempo real de pacientes y citas.
- **Diseño de Base de Datos Avanzado:** Estructura optimizada para manejar grandes volúmenes de historias clínicas sin pérdida de rendimiento.
- **UI/UX Profesional:** Interfaz limpia e intuitiva desarrollada con Tailwind CSS para una experiencia de usuario sin distracciones.

## 🧠 Desafíos Técnicos Resueltos

- **Integración Fullstack:** Sincronización perfecta entre el estado de Vue y el backend en Golang.
- **Seguridad:** Implementación de flujo de autenticación seguro para proteger datos sensibles de pacientes.
- **Optimización de Carga:** Estructura modular para asegurar un renderizado rápido en el cliente.

---

## 📋 Requisitos Previos

- Node.js (v18+)
- npm o yarn

## 🚦 Instalación y Ejecución

```bash
# Clonar el repositorio
git clone https://github.com/Oscar-Medrano/Frontend-Consultorio-Medrano.git

# Navegar al directorio del proyecto
cd Frontend-Consultorio-Medrano

# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm run dev

# Compilar para producción
npm run build
```

> **Nota:** Para la ejecución local, se requiere un archivo `.env` con las variables de entorno correspondientes. El acceso a la API de desarrollo está restringido.
